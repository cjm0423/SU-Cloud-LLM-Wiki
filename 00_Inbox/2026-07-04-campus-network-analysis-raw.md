## 1. 전체 물리 경로

```
운영계 서버 (.179) ─┐
                    ├─→ HPE 1820 ─→ 72번 벽포트 ─→ 415호 랙 ─→ 인터넷
개발계 서버 (.180) ─┘
```

- **405호 조교실**: 서버 2대 + HPE 1820 스위치가 위치하는 곳
- **415호 랙**: 건물 백본 배선함. 교내망 구간(학교 자산)과 KT 인입 구간(KT 자산)이 함께 있음
- 인터넷 이후 구간(KT망 내부)은 우리가 관리할 필요 없음

## 3. 415호 랙 구성 (2026-07-04 현장 확인)

### 3-1. 랙 구성 (위 → 아래)

- 데이터 흐름 (각 호실 → 인터넷 방향)
    
    ```bash
    각 호실 벽포트(72번 포함) 
        ↓
    패치패널 (AMP/NETCONNECT, 호실별 라벨)
        ↓
    교내망 구간 — DASAN V2624G-POES / Juniper EX3300
        (여러 호실 트래픽을 한 곳으로 모으는 역할)
        ↓
    KT 인입 구간 — olleh 스위치 → PREMIER 스위치 → PoE 인젝터군 → Ubiquoss → HPE 1820(415호) → DASAN V1824 R3 라우터
        (KT가 준 장비들, 마지막에 라우터가 실제 게이트웨이 역할)
        ↓
    광분배반 (VSOF-FDF-12C)
        (건물 외부에서 들어오는 KT 광케이블이 종단되는 지점)
        ↓
    인터넷 (KT 백본)
    ```
    
    ### 각 구간 역할
    
    **① 교내망 구간 (학교 자산)**
    
    - 각 호실 포트에서 온 트래픽이 패치패널에서 모여서 DASAN V2624G-POES, Juniper EX3300 같은 학교 소유 스위치로 집선됨
    - EX3300의 `Xe-0/0/1`, `Xe-1/0/4` 광모듈은 여러 스위치 간 10G로 빠르게 묶어주는 업링크로 추정
    - 즉, 이 구간까지는 순수하게 "건물 내부 여러 호실을 하나로 모으는" 역할
    
    **② KT 인입 구간 (KT 자산)**
    
    - 학교 내부망에서 모인 트래픽을 실제 인터넷으로 내보내기 위한 KT 제공 장비들
    - olleh 스위치, PREMIER 스위치, PoE 인젝터들은 중간 전달/전원 공급 역할이고
    - 맨 끝의 **DASAN V1824 R3 라우터**가 실질적인 게이트웨이 — 여기서 우리 쪽 IP(예: .179, .180)와 KT 네트워크 사이를 라우팅해주는 지점
    
    **③ 광분배반**
    
    - 건물 밖에서 들어오는 KT 광케이블이 물리적으로 종단되는 지점 (광케이블 ↔ 장비 간 연결부)
    - 데이터 자체를 처리하는 장비라기보다는 **광케이블 접속 패널** — KT 인입 구간 장비들과 외부 광회선을 이어주는 물리적 연결 지점
    
    ### 요약하면
    
    - **교내망 구간** = 우리 건물 안의 여러 호실을 모으는 구간
    - **KT 인입 구간** = 모인 트래픽을 실제 인터넷으로 내보내는 KT 장비 구간 (라우터가 핵심)
    - **광분배반** = 그 KT 장비와 외부 광케이블을 물리적으로 연결하는 접속점

**① 광분배반**

- SK telecom 광분배반 **VSOF-FDF-12C** (제조 2013.8, A/S 031-288-3484)
- class별 포트 매핑 (포스트잇 메모):

- Juniper **EX3300** — SFP 광모듈 4개(적2, 청2) 장착, `Xe-0/0/1`, `Xe-1/0/4` 라벨 (10G 업링크로 추정)
- AMP / NETCONNECT 패치패널 다단, 호실별 라벨 부착

**③ KT 인입 구간 (KT 자산)**

- HPE OfficeConnect 1820 스위치 (J9981A) — *415호 랙에도 동일 모델이 이미 있음, 405호 조교실 것과 혼동 주의*
- DASAN **V1824 R3** 라우터 (`HNA_ROUTER4501`)
- 48포트 스위치(SFP 49~51 포함) + 별도 24포트급 스위치(PORT 25 라벨)
